# web-usability

## Modules required:
* express
  * to create routes and APIs
* mongoose
  * to connect with mongoDB
* dotenv
  * to fetch environment variables
  
## API end-points

```
GET /participant/:id
response : participant details  
```

```
POST /participant/login
{
  email: "email id",
  password: "user password"
}
```
